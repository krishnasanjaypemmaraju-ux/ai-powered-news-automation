# ai-powered-news-automation
# 

A Python-based automation project that fetches the latest Indian and international news and delivers a structured daily email digest using secure API integrations.

---

## Features

- Fetches real-time news using NewsAPI  
- Covers both Indian and international headlines  
- Sends automated email digests via Gmail SMTP  
- Uses environment variables for secure credential handling  
- Clean, modular, and beginner-friendly Python code  

---

## Tech Stack

- Python  
- NewsAPI  
- SMTP (Gmail)  
- Requests  
- python-dotenv
